App UI is optimized for use on PC, but it works on mobile too.

"Reset" button is used to delete pending order (from the UI and from the variable that accumulates final price), and to clear THIS "rules section" before testing the app

To test the App create order using "order" buttons then complete the process by clicking "checkout" button on the nav bar.

If you dont input the order item or table number, you will recive error message and will have to repeat the ordering process.

After the order is successful, click "Login as owner" button to update the UI and see what items and where waiter needs to bring them.

You can also rate the products using stars, and after the app reloads, ratings will be pulled from local storage and update rating of the item.
